# NextAgent

这是一个AI审核系统，让一个AI审核另一个AI的内容，打破了AI长久以来幻觉严重的问题，虽然目前还非常简陋，没有GUI界面，但已实现基础的调用API，输入文本功能功能

还没有的功能（优先级从高到低）：

1.添加图形化界面

2.加入上传文件功能

3.适配AI喜欢用的Markdown格式文本


安装指南：

1.首先去[python官网](python.org)下载Python解释器

2.安装python解释器（一定要选**把Python添加到系统Path**）

3.下载本项目中的**NextAgent.py**

4.打开终端

5.切换到NextAgent.py所在的目录（**cd**命令）

6.终端中执行

```
python NextAgent.py
```

7.[下次启动本项目只需执行后三步]
